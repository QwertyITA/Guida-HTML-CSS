Comandi del CSS

nometag {} serve per modificare tutti gli elementi del tag che si sceglie di modificare
.nomeclasse serve per creare una classe che può assumere certe caratteristiche. La classe si può dopo assegnare a dei tag nell'html
#nomeid serve per creare un id che può assumere certe caratteristiche. Si possono combinare con le classe se in un tag fai class="XXX" id="YYY".

Attributi assegnabili ad ogni tag HTML

div {
height: 1500px; -> Indica l'altezza che un div deve assumere
width: 800px; -> Indica la larghezza che un div deve assumere
margin: 0px 0px 0px 0px; -> modifica il margine esterno (auto lo allinea al centro)
padding: 5px 5px 5px 5px; -> Indica quanto spazio lasciare all'interno di ogni div
border: solid black 50px; -> cambia l'aspetto del bordo
display: inline; -> Indica che il div deve stare sulla stessa riga del div precedente
}

p {
font-family: calibri; -> Indica il font che usa la scritta
font-size: 3058pt; -> indica la dimensione delle lettere
line-height: 4em -> indica lo spazio che verrà messo tra ogni riga
color: black; -> indica il colore del testo
}

.hover:hover {
background-color: #ededed; -> indica il colore di sfondo del link all'hovering del mouse
border-radius: 4px; -> indica il raggio di curvatura dello sfondo attivacon con l'hovering del mouse
}

a:link {
text-decoration: none; -> rimuove decorazioni (sottolineatura) al link non cliccato
color: black; -> colore del link non cliccato
}

a:visited {
color: black; -> colore link visitato
}
